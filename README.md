# Simple Text Generator with Streamlit

## Description
This project provides a simple web application using Streamlit to generate text based on a user-provided prompt. It leverages the `gpt2` model from the Hugging Face `transformers` library to create continuations of the input text.

## Features
- Interactive web interface for text input.
- Real-time text generation using a pre-trained language model.
- Customizable output length.
- Easy to set up and deploy.
